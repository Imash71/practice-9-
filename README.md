# Practice 9 - Pygame Games
## 📌 Project Description
This project was created as part of Programming Principles practice.  
It demonstrates basic game development using Python and Pygame, including graphics, animation, keyboard input, and audio playback.
---
## 🎮 Games Included
### 1. Mickey Clock 🕒
- Displays real system time
- Shows minutes and seconds
- Clock hands rotate based on time
- Includes numbered clock face (1–12)
### 2. Music Player 🎵
- Plays audio tracks using keyboard controls
- Controls:
 - P = Play
 - S = Pause
 - R = Resume
 - N = Next track
 - B = Previous track
 - Q = Quit
- Supports playlist system
### 3. Moving Ball Game 🔴
- Red ball moves using arrow keys
- Movement step: 20 pixels
- Boundary collision prevention (cannot leave screen)
---
## 🛠 Technologies Used
- Python 3
- Pygame library
